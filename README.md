# 📊 Sales Data Analysis Project

## 📌 Overview
This project performs basic data analysis on a real-world sales dataset using Python and pandas.  
The main goal is to understand sales performance, identify trends, and extract useful business insights.

---

## 🎯 Objectives
- Load and analyze sales data from a CSV file  
- Perform data cleaning (handle missing values and duplicates)  
- Explore dataset structure (rows, columns, data types)  
- Calculate key sales metrics  
- Generate a simple analysis report  

---

## 🛠️ Technologies Used
- Python  
- Pandas  

---

## 📂 Project Structure
Sales_Data_Analysis/

├── sales_analysis.py     # Main Python script  
├── sales_data.csv        # Dataset used for analysis  
├── analysis_report.md    # Project report  
├── requirements.txt      # Dependencies  
├── README.md             # Project documentation  

---

## ⚙️ How to Run the Project

### 1. Install Dependencies
pip install -r requirements.txt

### 2. Run the Script
python sales_analysis.py

---

## 🔍 Steps Performed

### 1. Data Loading
- Loaded dataset using pandas

### 2. Data Exploration
- Displayed first few rows  
- Checked dataset shape  
- Viewed column names and data types  

### 3. Data Cleaning
- Removed duplicate records  
- Filled missing numerical values using median  

### 4. Data Analysis
Calculated key metrics:
- Total Revenue  
- Average Sales  
- Highest Sale  
- Lowest Sale  
- Best Selling Product
