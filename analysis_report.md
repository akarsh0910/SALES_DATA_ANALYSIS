# Sales Data Analysis Report

## Project Overview
This project analyzes a simple sales dataset using Python and pandas.

## Steps Performed
1. Loaded sales data from a CSV file.
2. Displayed first few rows of the dataset.
3. Checked rows, columns, and data types.
4. Removed duplicate values.
5. Handled missing numerical values using median.
6. Calculated important sales metrics.

## Metrics Calculated
- Total Revenue
- Average Sales
- Highest Sale
- Lowest Sale
- Best-Selling Product

## Conclusion
The sales dataset was successfully cleaned and analyzed. The analysis helps understand total sales performance and identify the best-selling product.
